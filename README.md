# QR Scam Awareness & Reporting System
## Project overview
This Project is web-based platform developed to spread awarenss about QR code scams and provide users with a system to report fraud incidents.

QR code scams are increasing rapidly, especially in online payment Systems. This project aims to educate users and help then take preventive measures.

## Objectives
- Eduacate people about QR code fraud.
- Provide safety tips and preventive measures.
- Allow users to report scam incidents.
- Spread digital safety awareness.
